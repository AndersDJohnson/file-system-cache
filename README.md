# cache-fs
[![Build Status](https://travis-ci.org/philcockfield/cache-fs.svg)](https://travis-ci.org/philcockfield/cache-fs)

A super-fast, promise-based cache that reads and writes to the file-system.


## Run
    npm install
    npm start


## Test
    # Run tests.
    npm test

    # Watch and re-run tests.
    npm run tdd
